# 보은한복 홍보 웹사이트

보은한복 브랜드를 위한 현대적이고 아름다운 홍보 웹사이트입니다.

## 특징

- 🎨 **전통과 현대의 조화**: 한복의 전통 미학을 현대적인 웹 디자인으로 표현
- 📱 **완전 반응형**: 모바일, 태블릿, 데스크톱 모든 기기에서 완벽하게 작동
- ✨ **부드러운 애니메이션**: 스크롤 애니메이션과 인터랙티브 요소로 사용자 경험 향상
- 🎯 **직관적인 네비게이션**: 쉬운 탐색을 위한 고정 네비게이션 바
- 🖼️ **갤러리 섹션**: 제품과 브랜드를 소개하는 이미지 갤러리
- 📧 **문의 폼**: 고객 문의를 위한 접근 가능한 연락처 섹션

## 섹션 구성

1. **Hero Section**: 강렬한 첫인상을 주는 메인 배너
2. **About Section**: 보은한복의 브랜드 스토리와 특징
3. **Products Section**: 다양한 한복 제품 카테고리
4. **Gallery Section**: 한복을 입은 모델들의 갤러리
5. **Contact Section**: 문의 폼과 연락처 정보
6. **Footer**: 추가 정보와 소셜 미디어 링크

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (Vanilla JS)
- Google Fonts (Noto Sans KR, Noto Serif KR)

## 시작하기

1. 프로젝트 파일을 다운로드하거나 클론합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 또는 로컬 서버를 실행합니다:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server 필요)
   npx http-server
   ```

## 커스터마이징

### 색상 변경
`styles.css` 파일의 `:root` 변수를 수정하여 색상을 변경할 수 있습니다:

```css
:root {
    --primary-color: #c41e3a;      /* 주 색상 */
    --secondary-color: #1e3a8a;    /* 보조 색상 */
    --accent-color: #d4af37;        /* 강조 색상 */
}
```

### 이미지 추가
현재 이미지 플레이스홀더를 실제 이미지로 교체하려면:
1. 이미지 파일을 프로젝트 폴더에 추가
2. HTML에서 `.image-placeholder` 요소를 `<img>` 태그로 교체

### 콘텐츠 수정
`index.html` 파일에서 텍스트, 연락처 정보, 제품 정보 등을 수정할 수 있습니다.
